About
This program is a simple game made in C++ using the raylib library. The player controls a blue circle that must avoid/defeat the red enemy circles that pursue them around the screen as more and more of them spawn.

To run the program, open with the "notepad++.exe" and load in the program file. It's compiled with F6.

How to play:
Space = pause/unpause

AWSD directional controls
[A] = left
[W] = up
[S] = down
[D] = right

[R] = restart

[Mouse] = Direction of sword, a.k.a attack method

Goal:
Survive for as long as possible. New waves increase enemy count each wave
