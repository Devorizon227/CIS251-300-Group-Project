About
This program is a simple game made in C++ using the raylib library. The player controls a blue circle and must avoid/fight a red enemy circle that follows them around the screen.

To run the program, open with the "notepad++.exe" and load in the program file. It's compiled with F6.

How to play:
Space = pause/unpause

AWSD directional controls
[A] = left
[W] = up
[S] = down
[D] = right

[R] = restart

[Mouse] = Direction of sword, a.k.a attack method

Goal:
Survive for as long as possible
